# Module 12 - Docker Fundamentals

## Overview

This module introduces Docker and its role in containerizing applications. It covers Docker architecture, commonly used commands, Docker images, Dockerfiles, Docker Compose, networking, storage, and Kubernetes for container orchestration.

## Learning Outcomes

After completing this module, I was able to:

- Understand the purpose of Docker and containerization.
- Learn Docker architecture and its components.
- Execute basic Docker commands.
- Work with Docker images and containers.
- Understand Dockerfile creation.
- Learn Docker Compose for multi-container applications.
- Explore Docker networking and storage concepts.
- Understand the basics of Kubernetes and container orchestration.
