# Week 7 - Agile, DevOps, Docker, and Gen-AI Fundamentals

Welcome to Week 7! This week is about learning important IT skills in a simple way.

## What is in this folder?

- `Agile Methodology` - Notes and explanation about Agile, how teams work faster, and how to plan work.
- `Devops` - Notes and explanation about DevOps, which connects development and operations.
- `Docker` - Notes and explanation about Docker, a tool that helps run apps the same way everywhere.
- `Gen-AI Fundamentals` - Notes and explanation about generative AI basics.

## How to use this week

1. Start with `Agile Methodology` to learn how teams plan and deliver work step by step.
2. Read `Devops` to understand how coding and operations work together.
3. See `Docker` to learn how software can run inside containers.
4. Read `Gen-AI Fundamentals` to learn what generative AI is and how it works.

## Why it is useful

- Agile helps teams deliver work faster and stay organized.
- DevOps helps build and release software with fewer problems.
- Docker helps run apps in the same way on different computers.
- Generative AI helps create new content using smart models.

Use the notes in each folder to learn the main ideas clearly and easily.
