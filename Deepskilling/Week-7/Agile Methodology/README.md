# Module 13 - Agile Methodology

## Overview

This module introduces the Agile software development methodology and its core concepts. It explains Agile values, Scrum framework, estimation techniques, user stories, sprint planning, and team collaboration practices used in modern software development.

## Learning Outcomes

After completing this module, I was able to:

- Understand the fundamentals of Agile methodology.
- Explain the Agile Manifesto and its principles.
- Compare Agile and Waterfall development models.
- Learn Scrum roles, events, and artifacts.
- Understand Story Points and Planning Poker.
- Learn Sprint Planning and Velocity concepts.
- Create User Stories with Acceptance Criteria.
- Understand effective Agile team collaboration.
