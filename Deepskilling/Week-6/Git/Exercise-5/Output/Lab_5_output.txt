Module 10 - Exercise 5
Clean Up and Push to Remote Repository

1. Verified the repository status.

> git status

On branch main
nothing to commit, working tree clean

2. Listed all available branches.

> git branch -a

* main
  remotes/origin/main

3. Pulled the latest changes from the remote repository.

> git pull origin main

Already up to date.

4. Pushed local changes to the remote repository.

> git push origin main

Everything up-to-date

5. Verified the latest changes in the remote GitHub repository.

Result:

Successfully verified the repository status, listed all branches, synchronized the local repository with the remote repository using git pull, pushed the latest changes using git push, and confirmed that the changes were reflected in the remote repository.
