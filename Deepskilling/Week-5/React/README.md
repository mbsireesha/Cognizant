# ReactJS Hands-On Labs – Week 5 (Task 01 to Task 09)

This folder contains the first 9 ReactJS tasks from the Cognizant Deepskilling program. Each task focuses on a specific React concept, helping you build up your skills step by step.

## Task Summary

| Task | App Name | Concept Covered |
|------|----------|----------------|
| Task_01 | myfirstreact | Setting up React + Vite, Hello World |
| Task_02 | StudentApp | Creating and using multiple components |
| Task_03 | scorecalculatorapp | Passing data with Props |
| Task_04 | blogapp | Class components and lifecycle methods |
| Task_05 | cohort_styling_app | Styling with CSS Modules |
| Task_06 | TrainersApp | Navigation with React Router DOM |
| Task_07 | shoppingapp | State management in class components |
| Task_08 | counterapp | useState hook and event handling |
| Task_09 | cricketapp | ES6 features (arrow functions, spread, destructuring) |

## How to Run Any Task

```bash
# Step 1: Navigate to the task folder
cd Task_01

# Step 2: Install packages (first time only)
npm install

# Step 3: Start the app
npm run dev
```

Open `http://localhost:5173` in your browser to see the app running.

## Project Structure (Each Task)

```
Task_XX/
├── src/
│   ├── App.jsx          → Main component
│   ├── main.jsx         → Entry point
│   ├── index.css        → Global styles
│   └── components/      → Reusable components (if any)
├── index.html           → HTML template
├── package.json         → Dependencies
└── vite.config.js       → Vite configuration
```
