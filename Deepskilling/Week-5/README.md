# Week 5 – React Frontend Development (Task 01–09)

Welcome to Week 5! This week you will start building interactive web UIs using **ReactJS** — one of the most popular frontend libraries used in the industry today.

## What You Will Learn

- Setting up a React project using Vite
- Understanding JSX (JavaScript + HTML syntax)
- Working with components, props, and state
- Class-based vs functional components
- React Router for navigation between pages
- Handling events and user inputs

## How to Run Any Task

```bash
# Go into a task folder
cd Task_01

# Install dependencies (only needed once)
npm install

# Start the development server
npm run dev
```

Then open your browser and go to `http://localhost:5173`

> **Tip:** Each task builds on the previous one. Try completing them in order to get the best learning experience!

## Quick Concepts Recap

- **JSX** → Write HTML inside JavaScript
- **Props** → Pass data from parent to child component
- **State** → Data that changes and re-renders the UI
- **Component** → A reusable piece of UI (like a function)
