# Module 11 - DevOps and CI/CD Fundamentals

## Overview

This module provides an introduction to DevOps principles and the CI/CD pipeline. It explains how DevOps practices improve collaboration between development and operations teams while enabling faster, more reliable software delivery through automation.

## Learning Outcomes

After completing this module, I was able to:

- Understand the basic concepts of DevOps.
- Explain the purpose of Continuous Integration (CI).
- Describe the role of Continuous Delivery (CD).
- Compare Continuous Integration and Continuous Delivery.
- Recognize commonly used DevOps tools.
- Understand the DevOps workflow and its advantages.

## Topics Covered

- Introduction to DevOps
- Objectives and Benefits of DevOps
- Continuous Integration (CI)
- Continuous Delivery (CD)
- Continuous Deployment
- Difference between CI and CD
- Popular DevOps Tools
- DevOps Lifecycle
