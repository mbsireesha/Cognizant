# Week 6 - Git, CI-CD, and React

Welcome to Week 6! This week is about learning how to use Git, understand CI/CD concepts, and practice React work.

## What is in this folder?

- `Git` - Contains exercise folders where you practice Git commands, Git demos, and Git outputs.
  - `Exercise-1` to `Exercise-5`
- `CI-CD` - Contains notes and a README that explain DevOps and CI/CD ideas in a simple way.
- `React` - Contains React tasks from Task 10 to Task 19, including frontend practice with components and Vite.

## How to use this week

1. Open the `Git` folder to review the Git exercises.
2. Read the CI/CD notes in the `CI-CD` folder to understand basic DevOps concepts.
3. Explore the `React` tasks to see project examples and practice building user interfaces.

## Why it is useful

- Git helps you save changes and work with code versions.
- CI/CD helps automate testing and deployment so code is safer and faster to release.
- React helps you build web apps with components and interactive pages.

Keep the files in order, and use each folder’s README to guide your learning.
